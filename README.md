# Scirate-Bots
Bots for various platforms

# Discord Bot
### Installation 
- Follow this tutorial -> https://discordpy.readthedocs.io/en/stable/discord.html to create an Application on Discord.
- Add bot token to config.py 
  ```
  TOKEN=''
  PREFIX='!?'
  ```
- Docker
  ```
  docker build .
  docker run -i -t <IMAGE>
  ```
- Locally 
  ```
  python3 -B index.py 
  ```
